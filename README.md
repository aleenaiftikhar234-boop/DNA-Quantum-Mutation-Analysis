# Q-Mutate: Quantum DNA Mutation Analysis
A computational tool to predict genetic instability using the Schrödinger Equation.

### Overview
This project simulates **Proton Tunneling** in DNA base pairs. Traditional biology often ignores quantum effects, but this simulation proves that at 310K, protons can "jump" across hydrogen bonds, potentially leading to spontaneous mutations.

### Key Features
* **Sparse Hamiltonian Solver:** Uses efficient matrix algebra for high-speed calculation.
* **Thermal Population Modeling:** Accounts for human body temperature (Boltzmann distribution).
* **Mutation Risk Scoring:** Predicts the stability of G-C base pairs based on tunneling frequency.

### Built With
* Python
* Scipy (Sparse Linear Algebra)
* Matplotlib (Quantum Visualization)
